How to Use

  1. Select a cheque leaf number from your allocated 30 cheque leaves

  3. Enter the withdrawal amount (cannot exceed your balance)
  4. Enter the payee name exactly as it should appear on cheque
  5. Click Submit Request to send for admin approval
  6. Once used, a cheque leaf is removed from the dropdown
  7. Cancelled cheques return the leaf back to available
