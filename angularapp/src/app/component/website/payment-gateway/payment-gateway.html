Integration Guide

1. Create Order (Server-side)

POST /api/payment-gateway/orders/create
Content-Type: application/json

{
  "merchantId": "{{currentMerchant.merchantId}}",
  "amount": 1500.00,
  "description": "Premium Plan",
  "customerName": "John Doe",
  "customerEmail": "john@example.com"
}

2. Process Payment

POST /api/payment-gateway/pay
Content-Type: application/json

{
  "orderId": "ORD1234567890",
  "paymentMethod": "UPI",
  "payerAccount": "ACC987654321",
  "payerName": "John Doe"
}

3. Verify Signature (Critical)

POST /api/payment-gateway/verify-signature

{
  "orderId": "ORD1234567890",
  "transactionId": "PGTXN1234567890",
  "signature": "abc123...",
  "secretKey": "{{currentMerchant.secretKey}}"
}

// Signature = HMAC-SHA256(orderId + "|" + transactionId, secretKey)

4. Process Refund

POST /api/payment-gateway/refunds

{
  "transactionId": "PGTXN1234567890",
  "amount": 500.00,
  "reason": "Customer requested refund"
}
